Architecture

Built for Production

⚡
Real-Time Inference API
FastAPI server with async request handling, dependency injection, hot model reload, and Prometheus metrics — designed for <50ms P99 latency at scale.
FastAPI · uvicorn · Redis
🧠
Stacking Ensemble
Two-stage stacking: XGBoost + PyTorch TabTransformer as base learners trained with OOF predictions, Logistic Regression meta-learner with isotonic calibration.
XGBoost · PyTorch · Sklearn
🔬
Feature Engineering
87 features across 6 groups: temporal velocity (rolling windows 1h–7d), behavioral profiles, geo/network risk, device fingerprinting, and merchant graph signals.
Pandas · NumPy · Redis
📊
Experiment Tracking
Full MLflow integration for params, metrics, and artifact logging. Optuna HPO with TPE sampler and Hyperband pruner. Automatic model registry promotion.
MLflow · Optuna · DVC
🌊
Streaming Pipeline
Async Kafka consumer/producer for real-time transaction event processing. Configurable batch sizes, automatic offset commits, and dead-letter queue handling.
Kafka · aiokafka · asyncio
📡
Drift Monitoring
Population Stability Index (PSI), Kolmogorov-Smirnov tests, and Jensen-Shannon divergence for per-feature drift detection with Prometheus metrics + alerts.
Prometheus · Grafana · SciPy

Data Pipeline

End-to-End ML Pipeline

01
Data Ingestion
Batch (Parquet/CSV) + Kafka streaming. Schema validation with Great Expectations.
02
Validation
Column type checks, null rates, fraud rate bounds, referential integrity.
03
Feature Eng.
87 features across 6 groups. Fit on train, transform on all splits.
04
HPO
Optuna 100 trials, TPE + Hyperband. CV AUC-PR objective.
05
Train
2-stage stacking. OOF base learners → meta-learner on OOF preds.
06
Evaluate
AUC-ROC, AUC-PR, KS stat, Prec@K recall, fraud catch rate.
07
Register
Auto-promote to Production in MLflow if AUC-PR ≥ 0.85 quality gate.
